The application uses a browser; is open source. Please make a fork of the github repo and make your modification. By providing a pull request your modification will be merged into the main stream; so all users can enjoy the enhancement you make!
This app is based on different components:
  • WDSP ; An excellent DSP library by Warren Pratt, NR0V
  • Ported to Linux by John Melton, G0ORX
  • JNI java native interface to WDSP
  • JAVA
  • TOMEE Webserver using CDI
  • Angular (typescript and javascript)
  • Websockets and REST
  • W3.CSS Stylesheet
  • Maven

    • Enjoy remote working. Johan, PA3GSB.
